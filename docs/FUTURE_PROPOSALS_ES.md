# Roadmap: Evolución de Aetheris Core (OSS)

Aetheris Core seguirá evolucionando como una herramienta gratuita de alto rendimiento para desarrolladores Python y Web.

## Corto Plazo (Q2 2026)
- **Reflex Integration**: Plantillas nativas para usar el motor de física de Aetheris en proyectos de Reflex.
- **Auto-Clustering de Datos**: Algoritmos de centroides automáticos para etiquetar grupos de nodos en tiempo real.
- **HPC-lite en Web**: Optimización de buffer para manejar hasta 10,000 partículas en JS sin pérdida de FPS.

## Medio Plazo (Q3-Q4 2026)
- **Soporte para Pantallas Táctiles**: Refinamiento de la física de inercia y fricción para iPad/Androdid (interacción multisensorial).
- **Librería de Componentes Ampliada**: Inclusión de 15 nuevos widgets orientados a e-commerce (Galeria cinética, Botones magnéticos).
- **Mejoras en el Parser HTML**: Soporte nativo para layouts de Flexbox en el motor de física.
